My Projects

I have done the following projects at Gearbox academy

Keypad with servomotor

My first project at Gearbox with my peers. This is an Arduino project that uses a 4 by 4 keypad to move a servomotor by 90 degrees. It is a simple locking mechanism that requires a pin to unlock.

  • Project Documentation

Ultrasonic Sensor with RGB LED

Second project at Gearbox Academy. This project uses RGB LED to show different distances detected by an Ultrasonic Sensor. We placed an object at several distances from the ultrasonic sensor. The LED lit green, blue or red depending on the distance between the object and the ultrasonic sensor.

  • Project Documentation

ESP32 IOT Internet Control

I collaborated with a colleague on this project. The objective of this project is to learn how to move from an ESP32 to a cloud database and then from the internet back to the ESP32. ESP32 is preferred here since it is small, powerful, fast and can connect to WiFi. It works with 3.3V.

  • Project Documentation

Feedback Form using Python

In this project, I used anvil to create a simple feedback form containing a database, email functionality and deployed it on a website. Anvil allows the deployment of apps on the web without the need for HTML or CSS. The app records the user's name, email and feedback and sends it to my email
